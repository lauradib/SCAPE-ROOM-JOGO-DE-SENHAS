## Nome do Projeto

Escape Room: Encontre a Senha

## como funciona?

Voce acorda numa sala escura. Na parede tem um painel numerico com quatro espacos. Uma voz distante sussurra: "Acerte numeros para ser liberto." Voce tem 6 tentativas para descobrir a senha de 4 digitos e escapar de acordo com as dicas fornecidas.

## Como jogar?

1- Ter o Python instalado
2- Baixar o arquivo jogo_senha.py
3- Abrir o terminal na pasta do arquivo
4- Executar o comando:

```
python3 jogo_senha.py
```
