## Nome do Projeto

Escape Room: Encontre a Senha

## como funciona?

Voce acorda numa sala escura. Na parede tem um painel numerico com quatro espacos. Uma voz distante sussurra: "Acerte numeros para ser liberto." Voce tem 6 tentativas para descobrir a senha de 4 digitos e escapar de acordo com as dicas fornecidas.

## Como jogar?

1- Ter o Python instalado
2- Baixar o arquivo jogo_senha.py
3- Abrir o terminal na pasta do arquivo
4- Executar o comando:

```
python3 jogo_senha.py
```

## Exemplo de Interacao no jogo

```
Tentativa 1/6 - digite a senha de 4 digitos: 1234
Resultado: lugar certo / digito certo lugar errado / errado / errado
Dica: Um antigo relogio marca somente horas pares e todos os digitos seguem essa tradicao.
Tentativa 2/6 - digite a senha de 4 digitos: xxxx
VOCE ESCAPOU! Senha correta: xxxx
Conseguiu em 2 tentativas. Parabens!
```

## Conceitos Utilizados

- Variaveis
- Listas
- Entrada de dados com input()
- Condicionais if, elif, else
- Laco de repeticao while e for
