Nome do Projeto
Escape Room: Encontre a Senha
Historia do Jogo
Voce acorda numa sala escura. Na parede, um painel numerico com quatro espacos pisca. Uma voz distante sussurra: "Acerte numeros para ser liberto." Voce tem 6 tentativas para descobrir a senha de 4 digitos e escapar.
Como Executar

Ter o Python instalado no computador
Baixar o arquivo jogo_senha.py
Abrir o terminal na pasta do arquivo
Executar o comando:

python3 jogo_senha.py
